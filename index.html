xxdiff

This is the official home of xxdiff, a graphical file and directories comparator and merge tool. If you're longing the old days of SGI and xdiff, you've come at the right place.  xxdiff implements the same functionality and look-and-feel as xdiff, is provided as open source and adds many many useful new features. It runs on many popular unices, including IRIX, Linux, Solaris, HP/UX, DEC Tru64.

List of features

  • File comparison:
    • Look-and-feel similar to Rudy Wortel's/SGI xdiff
    • Default bindings understands emacs movement commands
    • Uses external diff program to compute differences: works with GNU diff, SGI diff and ClearCase's cleardif, and any other diff whose output is similar to those
    • Simple, order-independent 3-way diffs support (i.e. diff'ing 3 files, typically for merging)
    • Smart display of long filenames
    • Horizontal diffs highlighting, with optional ignoring of horizontal whitespace
    • Can spawn an editor session and refresh when one of the files has changed
    • Merge policing features:
      • cut-and-paste with annotations
      • filenames to clipboard (useful for long ClearCase names)
      • can save selected regions only
  • Merging files:
    • Easy selection of lines from both files (quick split/swap/merge feature)
    • Automatic merge selection, leaves conflicts as unselected
    • Can save unselected regions as conflicts with markers around them
    • View window of merged results
    • Global selection of unselected lines
  • Directory comparison:
    • Shallow and Recursive directory comparison
    • Can launch another xxdiff of selected files or directories under cursor
    • Optional ignore of file with changes (only file additions/removals are shown)
  • Misc.:
    • Overview area allows to see where changes occur in files
    • Full customization of all settings and colors through resource file, or optional Xrm resources
    • All resources can be specified on command line
    • Filename titles can be specified on command line, for CM version number display or other purpose
    • Desktop agnostic (i.e. will work equally well with KDE or Gnome)
xxdiff is tested daily in a merge reviewing/policing context by about 50 engineers at my workplace (discreet), and since this task is a significant part of their development process, many of the features are a direct result of suggestions from these people.

You will find other file comparison programs at dmoz. If you find a feature that is not available in xxdiff that you feel should be there, please let me know and I'll add it to the wishlist. The following major features are being actively worked on:

  • Configuration management integration (RCS, CVS, ClearCase)
  • Internationalization
Keep your eye on freshmeat for updates.
